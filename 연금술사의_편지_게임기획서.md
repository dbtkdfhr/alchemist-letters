## 1. 게임 개요

### 1.1 한 줄 소개

> 은퇴한 연금술사가 된 당신. 제자가 보낸 편지에 답장하며 연금술을 가르치는 선택지 기반 내러티브 게임.

### 1.2 장르

- **선택지 기반 내러티브** + **조합 퍼즐**
- 1인용 싱글플레이
- 추천 연령: 12세 이상

### 1.3 핵심 경험 (Core Experience)

- **편지를 읽는 경험**: 한 통의 편지가 주는 온도감과 이야기에 대한 기대
- **가르치는 재미**: 제자의 성장이 곧 나의 성취감으로 이어짐
- **발견의 재미**: 재료 조합을 실험하며 새로운 레시피를 찾아내는 쾌감
- **과거와의 대면**: 내 편지를 통해 과거의 내가 드러나는 구조

### 1.4 콘셉트 아트 (참고용, 텍스트로 표현)

```

│  ┌──────────────────────────────┐   │

│  │                                │   │
│  │  스승님께,                     │   │
│  │                                │   │
│  │  오늘 밤에 한 손님이          │   │
│  │  찾아왔습니다. 손바닥이        │   │
│  │  시뻘겋게 부어있었어요...      │   │
│  │                                │   │
│  │      - 제자 마르코 올림       │   │
│  └──────────────────────────────┘   │

│  [재료 보기]  [레시피]  [답장 쓰기]  │
└─────────────────────────────────────┘
```

## 2. 핵심 루프

### 2.1 한 챕터의 흐름

```
┌─────────────────────────────────────────────────────┐
│  ① 편지 도착 (제자의 사연 + 손님의 증상)            │
│     ↓                                                │
│  ② 증상 분석 — 무슨 문제인지 스스로 추론            │
│     ↓                                                │
│  ③ 재료 조합 — 실험하거나 기존 레시피 활용          │
│     ↓                                                │
│  ④ 답장 작성 — 선택지로 조언 + 레시피 전달          │
│     ↓                                                │
│  ⑤ 결과 확인 — 다음 편지에서 후일담披露             │
│     ↓                                                │
│  ⑥ (가끔) 회상 — 과거의 내 모습이 드러남            │
└─────────────────────────────────────────────────────┘
```

### 2.2 한 챕터 예상 시간: **20~30분**

| 단계 | 시간 | 설명 |
|------|------|------|
| 편지 읽기 | 2~3분 | 편지 내용 숙지, 분위기 흡수 |
| 증상 분석 + 재료 조합 | 10~15분 | 처음 보는 증상 → 실험 필요 |
| 답장 선택 | 1~2분 | 선택지 고민 |
| 결과 확인 | 2~3분 | 다음 편지에서 결과 확인 |
| **합계** | **15~23분** | |

> 이후 등장하는 신규 재료, 낯선 증상, 반복 손님으로 인한 변주로 챕터당 체감 시간이 일정하게 유지됨.

## 3. 편지 시스템

### 3.1 편지 = 게임의 모든 것

게임의 90%는 편지를 읽고 쓰는 인터페이스 위에서 이루어진다. 애니메이션이나 캐릭터 그래픽이 없으므로, **편지 UI 자체가 몰입도를 결정**한다.

### 3.2 편지 UI 구성 요소

```
┌───────────── 편지지 프레임 ──────────────┐
│                                           │
│  [봉투 아이콘]                         X │
│                                           │
│  ┌─ 편지지 (종이 텍스처) ──────────────┐ │
│  │                                       │ │
│  │  스승님께,                           │ │
│  │                                       │ │
│  │  오늘 찾아온 손님은 ...              │ │
│  │  [--- 스크롤 가능 ---]               │ │
│  │                                       │ │
│  │  증상을 어떻게 치료해야 할까요?      │ │
│  │                                       │ │
│  │          — 제자 마르코 드림          │ │
│  └───────────────────────────────────────┘ │
│                                           │
│  [연금술 책] [재료 상자] [답장 쓰기]      │
│                                           │
└───────────────────────────────────────────┘
```

### 3.3 편지 종류

| 종류 | 역할 | 빈도 |
|------|------|------|
| **일반 편지** | 손님 사연 + 증상 설명 | 매 챕터 기본 |
| **결과 편지** | 이전 답장의 후일담 | 직후 챕터에서 |
| **회상 편지** | 과거 스승의 기억 (플래시백) | 중요 챕터마다 |
| **긴급 편지** | 제한 시간 내 답장 필요 | 챕터당 1~2회 |
| **일기** | 제자의 개인적인 고민 (선택지 없음, 분위기용) | 가끔 |

### 3.4 손글씨 연출 (중요)

애니메이션이 없으므로 **정적인 연출**로 승부:

- **서체**: 손글씨 스타일 폰트 1~2종으로 통일
- **종이 질감**: 스캔한 종이 텍스처를 배경으로
- **번짐 효과**: 잉크가 번지는 듯한 테두리 효과 (CSS 또는 Godot Shader)
- **페이지 넘김**: 클릭 한 번에 다음 문장이 나타나는 **타자기 효과** 없이, **페이지 전환**만으로 텍스트 블록 표시
- **여백의 미**: 빈 공간을 활용한 감정 표현 (예: "죄송합니다"라는 말 뒤에 긴 여백)

### 3.5 편지 저장소

- 받은 편지는 모두 **편지 보관함**에 저장
- 언제든 다시 읽을 수 있음
- 회상 편지는 따로 분류되어 과거 조각 모음 역할

## 4. 연금술 시스템

### 4.1 설계 철학: "단순하지만 깊게"

- 복잡한 물리/화학 시뮬레이션 ❌
- 3~4개 속성의 조합으로 다양한 결과 도출 ✅
- 실험과 발견의 재미에 집중

### 4.2 재료 시스템

**기본 재료: 약 20종** (무료 아이콘으로 충분히 표현 가능)

| 분류 | 재료 예시 | 획득 시점 |
|------|-----------|-----------|
| 식물 | 달빛 이끼, 불꽃꽃, 그림자 버섯, 은빛 허브 | 초반 기본 |
| 광물 | 수정 가루, 철광석 분말, 월장석 | 초중반 |
| 동물성 | 용의 비늘(희귀), 불사조 깃털(희귀), 거미줄 | 중반 |
| 특수 | 별빛 물방울, 어둠의 진액 | 중후반 |
| 기본 용매 | 정제수, 달빛 오일, 화염 증류액 | 항상 보유 |

### 4.3 재료 속성 (5가지 기본 속성)

모든 재료는 1~2개의 **주 속성**과 1개의 **부 속성**을 가짐.

| 속성 | 역할 | 예시 재료 |
|------|------|-----------|
| 🔥 **열 (Ignis)** | 체온 상승, 염증, 활성 | 불꽃꽃, 화염 증류액 |
| ❄️ **냉 (Glacies)** | 진정, 마비, 보존 | 달빛 이끼, 정제수 |
| 🌱 **생 (Vita)** | 치유, 재생, 성장 | 은빛 허브, 달빛 오일 |
| 💀 **사 (Mortis)** | 독, 부식, 해체 | 그림자 버섯, 어둠의 진액 |
| ⚡ **기 (Spiritus)** | 균형, 촉매, 변환 | 별빛 물방울, 수정 가루 |

### 4.4 조합 시스템

**조합 방식**: 재료 2~3개를 슬롯에 넣고 "조합" 버튼

**조합 규칙** (플레이어가 실험을 통해 발견):
```
예시:
  달빛 이끼(❄️) + 불꽃꽃(🔥) = 대립 → 폭발/실패
  달빛 이끼(❄️) + 은빛 허브(🌱) = 상승 → 진정 연고
  은빛 허브(🌱) + 정제수(중성) = 약한 치유 물약
  은빛 허브(🌱) + 달빛 오일(❄️🌱) = 강력 치유 물약
```

**조합 결과 등급**:
| 등급 | 조건 | 결과 |
|------|------|------|
| 완벽 | 속성 3개 일치 + 희귀 재료 | 최고 효능, 추가 효과 |
| 성공 | 주 속성 일치 | 기본 효능 |
| 미미 | 부 속성만 일치, 혹은 하나만 일치 | 약한 효능 (제자가 "효과가 좀 약했어요") |
| 실패 | 대립 속성 포함 | 제자가 "뭔가 잘못된 거 같아요" |
| 대참사 | 3개 대립 | 폭발 사고 (제자가 다침) - 회복 필요 |

### 4.5 연금술 노트 (자동 기록)

- 성공/실패한 조합이 자동으로 기록됨
- 플레이어가 직접 이름을 붙일 수 있음 ("화상 연고", "수면제" 등)
- 언제든 열어볼 수 있는 일종의 도감
- 미발견 레시피는 "???"로 표시 → 발견 욕구 자극

### 4.6 증상 → 레시피 매핑 예시

| 손님 증상 | 필요한 효과 | 레시피 예시 |
|-----------|------------|-------------|
| 손바닥 화상 (🔥 과다) | 냉각 + 치유 | 달빛 이끼 + 은빛 허브 |
| 만성 피로 (기 부족) | 활성 + 보충 | 불꽃꽃 + 달빛 오일 |
| 불면증 (Spiritus 불안정) | 진정 + 균형 | 달빛 이끼 + 수정 가루 |
| 중독 (Mortis 과다) | 정화 + 해독 | 은빛 허브 + 정제수 (약함) |
| 화농성 상처 | 치유 + 부패 제거 | 은빛 허브 + 그림자 버섯(극소량) |

> 주요 과제: 플레이어가 단순히 정답을 맞추는 것이 아니라, "왜 이 재료면 효과가 있을까"를 **추론**하게 만드는 것.

## 5. 내러티브 구조

### 5.1 전체 구조: 3막 구성

**예상 총 플레이타임: 5시간** (15~20챕터, 챕터당 15~25분)

#### 제1막: "새싹" (챕터 1~6, 약 1.5시간)

> 제자는 갓 가업을 이어받은 초보다. 기본적인 연금술도 서툴다.

| 챕터 | 내용 | 주요 요소 |
|------|------|-----------|
| 1 | **첫 편지** — 제자 마르코의 첫인사. 기본 재료 소개. | 튜토리얼, 첫 조합 |
| 2 | **화상 입은 대장장이** — 가장 기본적인 치유. | 기본 레시피 |
| 3 | **잠 못 드는 귀부인** — 수면제 필요. | 첫 추론 필요 |
| 4 | **마르코의 실수** — 재료를 잘못 다뤄 사고. 제자가 자책. | 선택지 분기 시작 |
| 5 | **첫 회상** — 과거 스승이 제자였을 때. 같은 실수를 했다. | 과거 조각 #1 |
| 6 | **소문** — 마르코의 가게가 조금씩 유명해짐. | 1막 마무리 |

#### 제2막: "성장" (챕터 7~12, 약 2시간)

> 제자가 성장하고, 더 어려운 사례가 들어온다. 스승의 과거가 조금씩 드러난다.

| 챕터 | 내용 | 주요 요소 |
|------|------|-----------|
| 7 | **의문의 손님** — 낮에 나타나지 않는 손님. 뱀파이어? | 희귀 재료 등장 |
| 8 | **마르코의 자신감** — 제자가 스스로 조합을 시도. 선택지로 칭찬/질책 가능. | 분기 영향 시작 |
| 9 | **회상: 스승의 과오** — 과거 스승이 연금술로 누군가를 해친 기억. | 과거 조각 #2 |
| 10 | **전염병 소문** — 마을에 수상한 병이 돌기 시작. | 긴급 편지 |
| 11 | **마르코의 제안** — "제가 직접 가서 치료해도 될까요?" | 선택지 (가게 vs 직접) |
| 12 | **낯선 편지** — 마르코가 아닌 다른 누군가에게서 온 편지. | 2막 클리프행어 |

#### 제3막: "회복" (챕터 13~18, 약 1.5시간)

> 과거와 대면하고, 마르코는 독립한다.

| 챕터 | 내용 | 주요 요소 |
|------|------|-----------|
| 13 | **과거의 그림자** — 과거 스승이 저지른 일의 결과가 현재에 나타남. | 과거 조각 #3 |
| 14 | **마르코의 선택** — 제자가 스스로 결정을 내려야 하는 상황. | 통제 불가, 결과 지켜보기 |
| 15 | **마지막 가르침** — 스승이 제자에게 전하는 마지막 레시피. | 최종 레시피 |
| 16 | **독립 선언** — "스승님, 이제 저 혼자서도 할 수 있을 거 같아요." | 감정적 peak |
| 17 | **마르코의 첫 독립 편지** — 제자가 스스로 해결한 사례를 자랑. | 성장의 증거 |
| 18 | **에필로그** — 최종 편지. 마르코의 가게는 번성한다. 과거의 스승은... | 엔딩 분기 |

### 5.2 주요 캐릭터

#### 마르코 (제자)
- **성격**: 성실하지만 소극적. 초반에는 실수가 많음.
- **성장 곡선**: 플레이어의 선택에 따라 달라짐
  - 자주 칭찬 → 자신감 있는 연금술사 (일부 퍼즐 자동 해결)
  - 자주 질책 → 신중하지만 결정 장애 (추가 퀴즈 필요)
  - 다정한 조언 → 감성적인 치유사 (회상 이벤트 증가)

#### 스승 (플레이어, 당신)
- 본명은 게임 내에서 밝혀지지 않음. "스승님"으로만 불림.
- 과거에는 뛰어난 연금술사였으나, **어떤 사건**으로 인해 은퇴함.
- 그 사건의 진상이 게임을 통해 조금씩 드러남.

#### 조연들

| 캐릭터 | 설명 | 비고 |
|--------|------|------|
| **엘레나** | 대장장이. 첫 번째 손님. 이후 마르코의 단골이 됨. | 재등장 |
| **레아 귀부인** | 불면증 손님. 상류층. 이후 마르코의 가게를 널리 알림. | 네트워크 |
| **수상한 여행자** | 제2막부터 등장. 정체 불명. 스승의 과거를 아는 듯한 암시. | 핵심 인물 |
| **약초상 할머니** | 과거 스승의 지인. 가끔 편지로 조언을 전해줌. | 멘토 역할 |

### 5.3 분기 시스템

선택지는 크게 3가지 축으로 설계:

| 축 | 극단 A | 극단 B | 예시 상황 |
|----|--------|--------|-----------|
| **가르침 스타일** | 엄격 (정확함 우선) | 다정함 (실수 용인) | "레시피가 틀렸어" vs "괜찮아, 다음엔 맞출 거야" |
| **위험 감수** | 안전한 치료 | 과감한 실험 | "기존 레시피를 써라" vs "새로운 조합을 시도해 봐" |
| **과거와의 관계** | 과거를 숨김 | 과거를 고백 | "그 일은 잊어라" vs "사실 나도..." |

**분기의 실제 영향**:
- 엔딩 3종 (마르코의 독립 성공/안정/실패)
- 일부 챕터의 난이도 변화 (친절하게 가르치면 제자가 더 빨리 성장 → 후반 난이도 하락)
- 회상 편지의 양 (과거를 숨기면 회상 적음, 열면 많음)
- 재등장 인물의 반응 차이

## 6. 진행도 & 플레이타임

### 6.1 총량 계산

| 구성 요소 | 개수 | 단위 시간 | 합계 |
|-----------|------|-----------|------|
| 메인 챕터 | 15~18개 | 15~25분 | 4~5시간 |
| 선택지 분기 재생성 | 2~3회차 고려 | 70% 시간 | (선택) |
| 레시피 실험/탐험 | — | 플레이 중 포함 | — |

> 1회차 클리어에 **4~5시간** 목표. 선택지가 다른 2회차는 약 3시간.

### 6.2 후반으로 갈수록 해야 할 일 증가

| 구간 | 편지 읽기 | 증상 분석 | 재료 조합 | 답장 쓰기 |
|------|-----------|-----------|-----------|-----------|
| 초반 (1~5) | 5분 | 2분 | 5분 (가이드) | 1분 |
| 중반 (6~12) | 3분 | 5분 | 10분 | 2분 |
| 후반 (13~18) | 4분 | 3분 | 5분 (보유多) | 3분 (분기 깊음) |

> 초반은 편지 읽기에 치중, 중반은 조합 퍼즐에 집중, 후반은 선택지 고민이 깊어짐. 자연스러운 페이스 변화.

### 6.3 반복 플레이 요소

- 다른 선택지로 다른 엔딩
- 1회차에 못 찾은 레시피 발견
- "엄격한 스승" vs "다정한 스승" 경험 차이
- 특정 조건에서만 등장하는 숨은 편지

### 9.3 사운드 연출 포인트

- **조합 화면으로 전환 시 BGM 전환** — "실험실" 분위기로
- **중요한 선택지 앞에서는 BGM이 멈춤** — 2초 무음 후 재생 → 선택의 무게감
- **회상 장면**: 현재 BGM이 서서히 페이드아웃되고, 머플러 피아노가 fade-in

## 10. Godot vs 웹 비교

### 10.1 결정 기준표

| 항목 | Godot (4.x) | 웹 (HTML/CSS/JS) |
|------|-------------|-------------------|
| **개발 난이도** | ★★★☆☆ | ★★☆☆☆ |
| **배포 난이도** | 중 (실행 파일 빌드) | 하 (링크만 주면 끝) |
| **편지 UI 구현** | Control 노드로 가능 | CSS가 훨씬 자유로움 |
| **손글씨/종이 연출** | 제한적 (Shader 필요) | CSS 필터/텍스처로 풍부 |
| **사운드 관리** | AudioStreamPlayer로 간편 | Web Audio API (약간 복잡) |
| **저장/불러오기** | ResourceSaver | localStorage (매우 간단) |
| **이식성** | PC/모바일/콘솔 | 브라우저만 있으면 어디서나 |
| **확장 가능성** | 높음 (2D 게임 전반) | 낮음 (텍스트 게임에 최적) |

### 10.2 이 게임에 더 적합한 쪽은?

**→ 웹 (HTML/CSS/JS) 추천**

**이유**:
1. **이 게임의 90%는 UI/텍스트** — CSS가 Godot의 Control 노드보다 편지/종이 연출을 훨씬 자유롭고 쉽게 구현 가능
2. **로컬 저장(localStorage)** 으로 저장 시스템이 단 3줄이면 끝
3. **배포가 가장 쉬움** — HTML 파일 하나, 혹은 GitHub Pages로 무료 호스팅
4. **구현이 가장 빠름** — 한글 폰트 적용, 종이 텍스처, 페이지 전환 모두 CSS로 10분

**단, Godot을 선택해야 하는 경우**:
- 추후 본 게임을 더 복잡한 2D 게임으로 확장할 계획이 있다면
- PC 실행 파일로 배포하고 싶다면
- 더 정교한 사운드 관리(버스, 이펙트 등)가 필요하다면

### 10.3 하이브리드 전략 (추천)

1. **1차 프로토타입**: 웹 (HTML/CSS/JS) 으로 챕터 1~3까지 먼저 제작
2. **플레이테스트**: 반응 보고 게임성 검증
3. **전환 결정**: 확장이 필요하면 Godot으로 포팅, 아니면 웹으로 완성

> 텍스트 기반 게임은 **웹으로 충분하다**는 게 내 결론. Godot의 강점(물리 엔진, 애니메이션, 복잡한 게임 로직)을 거의 활용하지 않기 때문.

**편지 1 — 마르코의 첫인사**

> 스승님께,
>
> 오랜만에 편지를 드립니다. 그동안 잘 지내셨는지요?
>
> 저는 드디어 가게를 열었습니다. 대장장이 엘레나 님이 첫 손님이셨어요.
> "손바닥이 너무 뜨겁고 빨개요. 쇠를 만지다가 그만..." 하시더군요.
>
> 불꽃꽃을 으깨서 바르면 낫는다고 해서 그렇게 했는데...
> 다음 날 엘레나 님이 화가 나서 다시 오셨습니다. 손바닥이 더 부어있었어요.
>
> 제가 뭘 잘못한 걸까요? 불꽃꽃은 열을 내리는 약재 아니었나요?
>
> 가르침을 부탁드립니다.
>
>     — 제자 마르코 드림

### A.2 플레이어의 사고 과정 (의도된 흐름)

1. **증상**: 손바닥 화상 = 열(🔥) 과다 상태
2. **마르코의 실수**: 불꽃꽃(🔥)을 발라서 열을 더 가중시킴
3. **해결 방안**: 냉각(❄️) + 치유(🌱) 필요
4. **가능한 조합**: 달빛 이끼(❄️) + 은빛 허브(🌱) = 진정 연고 ✅
5. **답장 선택지**:
   - A: 불꽃꽃이 왜 틀렸는지 설명하고 정확한 레시피 전달 (엄격)
   - B: "실수는 누구나 한다"며 위로하고 레시피 전달 (다정)
   - C: 직접 레시피만 보내고 설명은 생략 (귀찮음)

### A.3 선택지 결과 (다음 챕터에서 반영)

| 선택 | 마르코의 반응 (다음 편지) | 효과 |
|------|--------------------------|------|
| A (엄격) | "죄송합니다. 다시는 실수하지 않겠습니다." | 마르코가 약간 위축되지만 정확도 상승 |
| B (다정) | "스승님 감사합니다! 자신감이 생겼어요!" | 마르코의 자신감 상승, 실험성 증가 |
| C (생략) | "설명을 더 듣고 싶었는데... 그래도 효과는 좋았어요." | 마르코가 약간 섭섭해함 |

## 부록 B: 추가 시스템 아이디어 (우선순위 낮음, 있으면 좋은 것)

- **타자기 연출 (선택)**: 특정 중요한 편지에만 한 글자씩 나타나는 효과 — 과도하게 사용 ❌
- **편지지 커스터마이즈**: 엔딩 후 해금 요소로 다른 종이 질감/봉투 색상
- **레시피 퍼즐 힌트 시스템**: 3번 이상 실패 시 "달빛 이끼가 어떨까?" 식의 힌트
- **속도 선택**: 편지 내용을 한 번에 다 보거나, 한 문장씩 넘기거나 선택 가능
- **봉투 미니게임**: 아주 가끔 편지가 봉투에 담긴 연출 (개봉할 때 긴장감)
